## Tech Stack

 - vue.js 2
 - @vue/cli 3
 - vuex
 - vue-router
 - libraries:
    - axios
    - lodash
    - bootstrap-vue UI
    - validator
 - Cloud Services
    - firebase (Google Cloud)
    - mailgun
    - StyleCI
    - Travis CI
    - Github Oauth
    - Google Oauth
 - API consumed:
    - Youtube API
    - Crypto Compare API
